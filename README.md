# 🎬 MeDit - CapCut-style Video Editor

This is the **lightweight, browser-based video editor** made with just HTML + JavaScript.  
No server required — fully offline and real exporting supported using ffmpeg.wasm.

## 🔗 Project Highlights

- ✅ No server — works by just opening `index.html`
- 🎨 Video effects: Grayscale, Invert, Brightness, Contrast
- 🎥 Real `.mp4` exporting in browser
- 💻 Works on desktop & mobile (use modern browser)
